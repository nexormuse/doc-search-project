# Week 02 학습 기록

## 오늘 학습한 내용
* 2주차 과제의 기능 1부터 기능 3까지 진행했다.
* 기술 문서의 텍스트를 검색에 활용할 수 있도록 소문자 변환, 특수문자 제거, 중복 공백 정리 등의 전처리 과정을 구현했다.
* 두 벡터의 내적과 노름을 이용해 코사인 유사도를 NumPy로 직접 계산하는 함수를 구현했고, 같은 벡터·직교 벡터·영벡터를 이용해 동작을 확인했다.
* 또한 질문과 문서에서 겹치는 단어의 개수를 점수로 사용하는 키워드 Baseline 검색을 구현했다.

## 구현한 기능
* `content` 결측치 제거
* `preprocess()` 텍스트 전처리 함수 구현
* `content_clean` 컬럼 생성
* `cosine_similarity_numpy()` 직접 구현
* 벡터 크기 불일치 예외 처리
* 영벡터 입력 시 0으로 나누는 문제 방지
* `keyword_search()` 키워드 검색 구현
* 질문과 문서의 단어 교집합을 이용한 점수 계산
* 점수 기준 Top-3 문서 반환

## 실행 결과
* 같은 벡터의 코사인 유사도: 1.0000
* 직교 벡터의 코사인 유사도: 0.0000
* 영벡터가 포함된 경우: 0.0000
* Keyword Baseline 검색에서 Top-3 결과가 정상적으로 출력됨

## 막힌 부분
* 코사인 유사도를 계산할 때 벡터의 크기가 0이면 분모가 0이 될 수 있다는 점과, 키워드 검색에서 질문과 문서를 어떤 방식으로 비교해야 하는지가 처음에는 헷갈렸다.

## 해결 과정
* 두 벡터의 노름을 먼저 계산하고 하나라도 0이면 유사도 0.0을 반환하도록 처리했다.
* 키워드 검색에서는 질문과 문서를 각각 단어 집합으로 변환하고, 두 집합의 교집합 크기를 검색 점수로 사용했다.
